# LooPay - 

A modern cross-border payment application designed specifically for Kenyans both local and in the diaspora. Built with React, TypeScript, and Tailwind CSS.

## 🚀 Features

- **Multi-currency support** - Send and recieve money in KES, USD, EUR, and more
- **M-PESA integration** - Seamless mobile money transfers
- **Bank account linking** - Connect your local and international accounts
- **Real-time exchange rates** - Get the best rates for your transfers
- **Bill payments** - Pay utilities, school fees, and more
- **Split bills** - Share expenses with friends and family
- **Crypto support** - Buy and sell cryptocurrencies

## 🛠️ Tech Stack

- **Frontend**: React 18 + TypeScript
- **Styling**: Tailwind CSS + shadcn/ui
- **State Management**: TanStack Query + React Hook Form
- **Routing**: React Router DOM
- **Icons**: Lucide React
- **Build Tool**: Vite
- **Linting**: ESLint + TypeScript ESLint

## 🏃‍♂️ Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/looppay-diaspora-connect.git

# Navigate to the project directory
cd looppay-diaspora-connect

# Install dependencies
npm install

# Start development server
npm run dev
```

### Available Scripts

```bash
npm run dev          # Start development server
npm run build        # Build for production
npm run preview      # Preview production build
npm run lint         # Run ESLint
npm run lint:fix     # Fix ESLint issues
npm run type-check   # Run TypeScript type checking
npm run clean        # Clean build artifacts
```
